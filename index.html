Cloud sync setup

Free + transferable template mode: leave the built-in config blank, then each new user creates their own free Supabase project and pastes their own Project URL and anon/public key here. This keeps the dashboard from being tied to your account.
This information is saved in this browser only. To transfer the template to another user, send them this HTML file, the SQL script, and the setup guide. They should use their own Supabase account/project for their own data.
